# blkid命令

blkid程序是使用libblkid库的命令行界面。它可以确定块设备持有的内容类型（例如文件系统、交换），以及内容元数据中的属性（令牌、名称=值对）（例如LABEL或UUID字段）。
blkid有两种主要的操作形式：要么搜索具有特定NAME=值对的设备，要么显示一个或多个设备的NAME=值配对。
