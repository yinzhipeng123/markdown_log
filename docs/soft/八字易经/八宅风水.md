

## **1️⃣ 西四命（乾、坤、兑、艮）**

| 类型       | 方位 | 运势作用                            |
| ---------- | ---- | ----------------------------------- |
| **四吉方** | 正西 | **生气** → 事业、财富、动力         |
|            | 西北 | **延年** → 人际关系、婚姻、家庭和谐 |
|            | 西南 | **天医** → 健康、疾病减少           |
|            | 东北 | **伏位** → 学业、修养、安稳         |
| **四凶方** | 正东 | **五鬼** → 麻烦、小人、灾祸         |
|            | 东南 | **祸害** → 健康、感情、事业受影响   |
|            | 正南 | **绝命** → 严重阻碍、破财、健康受损 |
|            | 正北 | **六煞** → 意外、官非、健康问题     |

## **2️⃣ 东四命（震、巽、坎、离）**

| 类型       | 方位 | 运势作用                                              |
| ---------- | ---- | ----------------------------------------------------- |
| **四吉方** | 正东 | **生气** → 事业、财富、动力                           |
|            | 东南 | **延年** → 人际关系、婚姻、家庭和谐                   |
|            | 正南 | **天医** → 健康、疾病减少                             |
|            | 正北 | **伏位** → 学业、修养、安稳                           |
| **四凶方** | 正西 | **五鬼** → 麻烦、小人、灾祸                           |
|            | 西北 | **延年** → 健康、事业不顺（西四命吉方对东四命是凶方） |
|            | 西南 | **绝命** → 严重阻碍、破财、健康受损                   |
|            | 东北 | **六煞** → 意外、官非、健康问题                       |





| 卦   | 象意           | 五行 | 性质偏向                 |
| ---- | -------------- | ---- | ------------------------ |
| 乾   | 天、刚健、主动 | 金   | 事业、权力、外向、动力强 |
| 坤   | 地、柔顺、承载 | 土   | 家庭、安稳、孕育、包容   |
| 兑   | 泽、喜悦、交流 | 金   | 社交、沟通、快乐、轻松   |
| 艮   | 山、止、稳重   | 土   | 学习、修养、静心、沉稳   |
| 震   | 雷、行动、奋发 | 木   | 创新、冒险、成长         |
| 巽   | 风、顺、渗透   | 木   | 学习、适应、人际         |
| 坎   | 水、险、流动   | 水   | 智慧、考验、挑战         |
| 离   | 火、明亮、热情 | 火   | 光明、感情、动力         |





**乾卦（天）** → 主外向、主动、事业运强

- 吉方放 **办公室/书房/事业重点房间** 可以增强行动力和外向运势

**坤卦（地）** → 主柔顺、稳定、家庭和谐

- 吉方放 **卧室或家庭活动区** 可以增强家庭感情和安全感

**兑卦（泽）** → 主交流、喜悦、人际

- 吉方放 **客厅/会客区** 有利社交、招待客人、活跃气场

**艮卦（山）** → 主静止、修养、学习

- 吉方放 **书房/静坐区/修炼区** 有利学习、思考、静心
