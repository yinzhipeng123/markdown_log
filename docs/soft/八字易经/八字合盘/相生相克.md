#### 相克

相克是 **隔一个克一个**。

口诀：

**木克土
 土克水
 水克火
 火克金
 金克木**

为什么这样？

| 克     | 解释     |
| ------ | -------- |
| 木克土 | 树根破土 |
| 土克水 | 土挡水   |
| 水克火 | 水灭火   |
| 火克金 | 火融金   |
| 金克木 | 斧头砍树 |



#### 相合

相合在八字里主要有 **两类**：

1️⃣ **天干合（五合）**
2️⃣ **地支合（六合、三合等）**



十天干有 **5组固定组合**：

| 天干合  | 化什么 |
| ------- | ------ |
| 甲 + 己 | 合土   |
| 乙 + 庚 | 合金   |
| 丙 + 辛 | 合水   |
| 丁 + 壬 | 合木   |
| 戊 + 癸 | 合火   |



地支有 **六合**：

| 地支合 |
| ------ |
| 子 丑  |
| 寅 亥  |
| 卯 戌  |
| 辰 酉  |
| 巳 申  |
| 午 未  |



三合局

| 三合局   | 五行 |
| -------- | ---- |
| 申 子 辰 | 水局 |
| 亥 卯 未 | 木局 |
| 寅 午 戌 | 火局 |

**隔五合**

例如：

#### 相刑

##### 1 无恩之刑

**寅刑巳
 巳刑申
 申刑寅**

这是 **三刑循环**。

意思：

- 寅 → 巳
- 巳 → 申
- 申 → 寅

这种刑常被认为：

**冲突比较明显。**

------

##### 2 恃势之刑

**丑刑戌
 戌刑未
 未刑丑**

也是三角关系。

特点：

**互相不服气。**

------

##### 3 无礼之刑

**子刑卯**

这个只有一组。

传统命理叫：

**子卯无礼刑**

通常解释为：

- 情绪冲突
- 礼数不合

------

##### 4 自刑

四个地支会 **自己刑自己**：

| 地支 |
| ---- |
| 辰   |
| 午   |
| 酉   |
| 亥   |

例如：

**辰辰**

叫：

**自刑**

意思：

**自己和自己过不去。**
