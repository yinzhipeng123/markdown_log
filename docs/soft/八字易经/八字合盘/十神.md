

## 十神一览（先有个整体感觉）

| 类型 | 名称       | 本质         |
| ---- | ---------- | ------------ |
| 同类 | 比肩、劫财 | 和你一样的人 |
| 生我 | 正印、偏印 | 帮你的人     |
| 我生 | 食神、伤官 | 你输出的东西 |
| 克我 | 正官、七杀 | 约束你的人   |
| 我克 | 正财、偏财 | 你控制的资源 |



## 喜神（对你有利）

能让你“平衡、舒服、发展好”的五行或十神



## 忌神（对你不利）

 让你失衡、过度、出问题的



你要做的是：

- 多学习（印）
- 找资源支持（印）
- 找靠谱团队（比劫）
- 少冒险（财、偏财）
- 少硬扛压力（官杀
