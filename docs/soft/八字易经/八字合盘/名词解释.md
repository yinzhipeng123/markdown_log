



## 🌕 一、整体概念速览

| 名称     | 性质类型   | 说明一句话总结                                         |
| -------- | ---------- | ------------------------------------------------------ |
| **相同** | 和谐型     | 两个相同地支（如子子），力量增强、气场一致。           |
| **相生** | 和谐型     | 五行相生（木生火等），互相滋养、助力。                 |
| **相克** | 对立型     | 五行相克（金克木等），制约、冲突。                     |
| **六合** | 和谐型     | 特定地支配对组合成“合气”，情感温和、互助。             |
| **三会** | 和谐型     | 三个连续地支聚集成一气，合作强、气场一致。             |
| **五合** | 和谐型     | 天干之间的相合，代表理念相投、情感连结（作用在天干）。 |
| **暗合** | 潜在和谐型 | 没有直接六合，但隐藏气场暗合，有隐缘或暗助。           |
| **六冲** | 对立型     | 六对相冲（如子午冲），力量对抗、易变动。               |
| **相刑** | 矛盾型     | 三刑、六刑关系，暗中不合，带来压力或摩擦。             |
| **相害** | 误会型     | 表面和平、内里互损，代表误解或小人。                   |
| **相破** | 潜冲型     | 力量不大但破坏气场，代表破损、疏离。                   |

## 🌿 二、详细分类表

| 关系名           | 地支组合                                                     | 性质     | 命理含义                 | 常见象征               |
| ---------------- | ------------------------------------------------------------ | -------- | ------------------------ | ---------------------- |
| **相同**         | 如子子、丑丑                                                 | 同类     | 力量加强，气场一致       | 志趣相投、理念一致     |
| **相生**         | 五行相生（木→火→土→金→水→木）                                | 生助     | 互相滋养、扶持           | 成长、助缘、合作       |
| **相克**         | 五行相克（木克土、土克水、水克火、火克金、金克木）           | 克制     | 制衡、冲突               | 竞争、约束、挑战       |
| **六合**         | 子丑、寅亥、卯戌、辰酉、巳申、午未                           | 温和合   | 情感、合作、缘分深厚     | 婚缘、贵人、和气       |
| **三会**         | 亥子丑（水局）、寅卯辰（木局）、巳午未（火局）、申酉戌（金局） | 合化     | 同气相求、力量集中       | 团结、合作、方向一致   |
| **五合（天干）** | 甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火             | 和谐     | 天干合化，思想或命运连接 | 感情、事业合拍         |
| **暗合**         | 非正式六合，如卯与午、子与未等（不同命理流派略有不同）       | 潜合     | 暗中有缘，有时是隐性关系 | 暗恋、暗助、潜缘       |
| **六冲**         | 子午、丑未、寅申、卯酉、辰戌、巳亥                           | 强冲     | 冲动、变动、分离         | 冲突、搬家、变更、分手 |
| **相刑**         | 丑未戌（三刑）、寅巳申（三刑）、子卯（无恩之刑）、辰午酉亥（自刑） | 暗中不合 | 表面平静、内里不顺       | 难协调、暗斗、病灾     |
| **相害**         | 子未、丑午、寅巳、卯辰、申亥、酉戌                           | 误伤型   | 表面相安、内里相害       | 误会、被动伤害、小人   |
| **相破**         | 子酉、丑辰、寅亥、卯午、申未、巳戌                           | 破裂型   | 不合但非正冲，力量较轻   | 分歧、破损、失和       |

## 🔥 三、理解口诀（便于记忆）

| 类别         | 特征口诀                                |
| ------------ | --------------------------------------- |
| **和谐类**   | 生、合、会、同、暗合 → 缘分、助力       |
| **冲突类**   | 克、冲、刑、害、破 → 纠葛、阻力         |
| **潜伏类**   | 暗合、相害、相破 → 外平内异、暗流涌动   |
| **力量强弱** | 冲 > 刑 > 害 > 破（冲最明显、破最隐晦） |

## 🌙 四、在**合盘**（两人八字）中的解释

| 关系类型                             | 感情或人际象征                       |
| ------------------------------------ | ------------------------------------ |
| **六合 / 三会 / 暗合**               | 容易一见如故、互相吸引、有缘分或默契 |
| **相生 / 相同**                      | 理解对方、性格相容、互相助长         |
| **相冲 / 相克 / 相刑 / 相害 / 相破** | 意见相左、吸引中带矛盾、磨合困难     |
| **冲中有合**（比如六合又被冲）       | 爱恨交织，情感强烈但易起波折         |

------

## 🌸 五、小结一句话：

> 🌈 **合为缘，冲为变，刑为难，害为误，破为散，生为助，克为制。**
