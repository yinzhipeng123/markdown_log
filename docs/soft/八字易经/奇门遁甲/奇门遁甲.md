# **一、奇门遁甲是什么？**

奇门遁甲是一套**古代高级决策系统**，本质是把时间和空间编码，然后通过符号（天盘、地盘、八门、九星等）帮助古人判断：

- 做什么事情吉还是凶
- 哪个方向好
- 什么时候做更顺
- 事情趋势如何

它可以用于：
 **预测、决策、选时、选方向、风水、用兵、管理**。

你可以把它理解为：
 👉 **古代版的大数据 + 决策算法模型**。

------

# 🌟 **二、奇门遁甲的核心结构是什么？**

奇门遁甲的盘面主要由以下组成：

## **1. 三个盘：天、地、人**

- **天盘**：象征“天时”，力量最大
- **地盘**：象征“地利”，稳定不动
- **人盘（值使门）**：象征“人和”，你自己可以掌控的部分

三盘合在一起，就是“天时地利人和”的解读模型。

------

## **2. 九星（偏心理、趋势）**

例如：

- **天蓬**：水，疑、险、聪明
- **天任**：稳重、可靠
- **天禽**：中宫核心之星，万能
- **天英**：火，聪明、文采
- **天辅**：帮助、贵人
- **天冲**：冲动、迅速
- **天芮**：病气、脏乱
- **天柱**：阻碍
- **天心**：心思细腻、医术、智慧

------

## **3. 八门（奇门遁甲“最重要”部件）**

关键词如下（重点记）：

| 八门     | 代表含义               | 吉凶     |
| -------- | ---------------------- | -------- |
| **开门** | 开始、成功、谈判、考试 | 吉       |
| **休门** | 休息、财富、人缘、平稳 | 吉       |
| **生门** | 生意、财运、健康、增长 | 大吉     |
| **伤门** | 损伤、运动、外科、激烈 | 中凶     |
| **杜门** | 封闭、保守、隐藏       | 中性偏静 |
| **景门** | 文采、名气、外表、火象 | 吉       |
| **死门** | 阻碍、停滞、不顺       | 凶       |
| **惊门** | 惊吓、压力、变化       | 中凶     |

**一般认为最吉的：生、休、开、景四门。**

------

## **4. 三奇六仪（奇门名称的来源）**

**三奇（最吉利的）：**

- **乙奇**（阴木）
- **丙奇**（太阳火）
- **丁奇**（灯火）

**六仪（普通或偏凶）：**
 戊、己、庚、辛、壬、癸

丙、丁、乙位置好时就是“奇门遁甲中的黄金信号”。

# 🌟 三、奇门遁甲怎么学？

给你一个**入门路线图**：

------

## **第一步：理解奇门结构**

（你现在就在学这一部分）

重点掌握：

- 八门（必须会）
- 九星（理解即可）
- 八神（玄武、白虎等，初期可不急）
- 三奇六仪（认识即可）

------

## **第二步：学会“怎么起局”**

主要两种：

- **时家奇门（现在最常用）**
- **日家奇门（风水师用多）**

你可以使用手机 App 起局，不需要自己手算。

------

## **第三步：判断吉凶（核心）**

入门最常用的判断方法：

✔ 看事情落宫
 ✔ 看八门好不好
 ✔ 看宫位是否为吉格
 ✔ 看门、星、神、奇仪的组合
 ✔ 看用神受到克还是生
