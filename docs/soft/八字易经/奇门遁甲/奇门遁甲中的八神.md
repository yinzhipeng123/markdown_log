八神（也称八诈神）是奇门遁甲中极为重要的信息符号，代表了超越物质世界的高维能量、特殊信息和神助体系。它们虽然名为“神”，但更多是一种**能量状态、信息特质和吉凶属性的象征**。

### 核心总览：八神的阴阳序列

八神的排列顺序（值符、腾蛇、太阴、六合、白虎、玄武、九地、九天）并非随机，它暗合了事物从发展到终结的完整循环，也分阴阳：

- **值符、腾蛇、太阴、六合** 为 **“阳神”** ，多主开始、发展、显现。
- **白虎、玄武、九地、九天** 为 **“阴神”** ，多主变化、结果、隐藏。

### 八神详解

#### 1. 值符（天乙贵人）

- **定位**：**八神之首，最吉之神**。相当于当前时空的“值班统帅”或“核心力量”。
- **特质**：代表 **领导、贵人、核心、正统、权力、名望、主流价值观**。它到哪里，哪里就有庇佑和正气。
- 象意延伸：
  - [ ] 人物：上司、老板、官方人士、核心人物。
  - [ ] 事物：重要的事情、官方文件、品牌产品、高品质的东西。
  - [ ] 状态：稳健、可靠、正大光明。
- **应用**：预测时，临值符通常事情有贵人相助，或与领导、重要人物有关。择吉时，首选值符方。

#### 2. 腾蛇（虚诈之神）

- **定位**：**虚惊怪异，变化不定之神**。
- **特质**：代表 **缠绕、虚假、变化、欺骗、惊恐、灵异、心机、反复**。像蛇一样纠缠不清，难以捉摸。
- 象意延伸：
  - [ ] 人物：狡诈之人、精神病患者、心术不正者。
  - [ ] 事物：虚假信息、网络谣言、慢性疾病、噩梦、纠缠不清的感情或债务。
  - [ ] 状态：心神不宁、犹豫不决、事情反复。
- **应用**：预测时，临腾蛇主事情虚假、变化多端，或当事人心烦意乱。测病可能是怪病或心理疾病。

#### 3. 太阴（荫护之神）

- **定位**：**阴佑、密谋、庇护之神**。
- **特质**：代表 **阴私、密谋、策划、隐藏的庇护、细致的思考、女性贵人**。它像月光一样，不是太阳的直接照耀，而是间接的、隐藏的帮助。
- 象意延伸：
  - [ ] 人物：谋士、策划者、暗中帮助你的贵人、女性（尤其是能提供智力支持的）。
  - [ ] 事物：秘密计划、暗箱操作、精致的物品、隐匿之事。
  - [ ] 状态：深思熟虑、暗中进行、有福荫。
- **应用**：预测合作、谈判，临太阴利于暗中谋划。测事，可能有隐秘的助力。但若测公开之事，则可能“见不得光”。

#### 4. 六合（和合之神）

- **定位**：**和睦、合作、中介之神**。
- **特质**：代表 **合作、婚姻、交易、中介、团聚、包容**。它是人际关系、合作共赢的象征。
- 象意延伸：
  - [ ] 人物：媒人、中介、合作伙伴、夫妻、团队。
  - [ ] 事物：合同、协议、婚姻、多人聚会、与人打交道的事。
  - [ ] 状态：和谐、愉快、牵线搭桥。
- **应用**：预测婚姻、合作、谈判、求职，临六合是大吉之兆。出行访友也主顺利愉快。

#### 5. 白虎（凶煞之神）

- **定位**：**凶灾、伤损、威权之神**。 **大凶神，也主权威**。
- **特质**：代表 **伤害、凶灾、疾病、死亡、血光、道路危险、官方压力、威严、强大攻击力**。非常刚猛，凶性最强。
- 象意延伸：
  - [ ] 人物：敌人、暴徒、执法人员（如警察、军人，取其威严一面）、外科医生（取其开刀一面）。
  - [ ] 事物：交通事故、打架斗殴、手术、牢狱、激烈的竞争。
  - [ ] 状态：压力巨大、危险临头、性情刚烈。
- **应用**：预测时，临白虎主大凶，需极度谨慎，尤其注意安全。但对于军警、竞技体育等需要“杀气”的行业，白虎反而代表能力和威严。

#### 6. 玄武（盗骗之神）

- **定位**：**偷盗、欺骗、暗昧、智慧之神**。
- **特质**：代表 **偷盗、欺骗、暧昧、不明朗、投机、暗中的小人、狡猾、同时也主聪明才智**。它不像白虎那样明着来，而是暗中行事。
- 象意延伸：
  - [ ] 人物：小偷、骗子、小人、投机者、聪明但心术不正的人。
  - [ ] 事物：丢失财物、上当受骗、暧昧关系、走私、幕后交易。
  - [ ] 状态：混乱不清、真假难辨、有口难言。
- **应用**：预测财物，临玄武可能被盗或投资被骗。测感情，可能关系暧昧不清。测事，有小人暗中捣鬼。

#### 7. 九地（坚牢之神）

- **定位**：**稳固、静守、包容之神**。
- **特质**：代表 **大地、稳定、长久、缓慢、包容、蛰伏、基础扎实**。主静，利于防守、积累。
- 象意延伸：
  - [ ] 人物：沉稳的人、农民、地产商。
  - [ ] 事物：房地产、矿产资源、长期规划、慢性子的事、基础建设。
  - [ ] 状态：稳定发展、进展缓慢、潜力深厚。
- **应用**：预测发展，临九地主事缓，但基础稳。适合用于守成、囤积、学习打基础。不宜用于需要快速决断、开拓进取之事。

#### 8. 九天（威扬之神）

- **定位**：**高远、动态、张扬之神**。
- **特质**：代表 **天空、远大目标、变动、张扬、传播、升腾**。主动，利于进攻、传播、上升。
- 象意延伸：
  - [ ] 人物：有远大抱负的人、飞行员、传播者。
  - [ ] 事物：航空、出国、宣传广告、远行、高调行事。
  - [ ] 状态：志向高远、变动性强、声势浩大。
- **应用**：预测事业，临九天利于开拓市场、扬名立万。出行远游，临九天也吉。但若测需要低调、保密的事，临九天则不吉。
