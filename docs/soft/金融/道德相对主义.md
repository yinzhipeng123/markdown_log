[道德相对主义 - 维基百科，自由的百科全书](https://zh.wikipedia.org/wiki/道德相对主义)



同一件事，在不同体系里可能：

- 被认为是对的
- 或被认为是错的

而**没有一个“绝对正确”的标准可以统一裁决**。



例如：

- 道德来自文化
- 不同文化有不同规则
- 对错取决于个人
- 对我来说是对的就可以

| 观点         | 核心             |
| ------------ | ---------------- |
| 道德绝对主义 | 有统一的对错标准 |
| 道德相对主义 | 对错依赖背景     |
| 道德虚无主义 | 根本没有对错     |
