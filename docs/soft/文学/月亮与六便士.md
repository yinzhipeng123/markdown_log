# **《月亮与六便士》故事梗概（毛姆）**

小说灵感来自画家高更的一生。主人公 **查尔斯·思特里克兰德（Charles Strickland）** 是一个伦敦证券经纪人、中产阶级、家庭和事业都稳定的“体面人”。
 但在 40 多岁时，他突然抛妻弃子，离开伦敦，前往巴黎学画。

### **1. 抛弃平凡生活，走上孤独创作之路**

思特里克兰德并不是为了追求浪漫或名利，而是被一种强烈到近乎病态的“内在冲动”驱使：**必须绘画**。
 他漠视世俗、无情冷酷，对家庭、友谊、道德完全不在意，只专注于自己的艺术。

### **2. 艰难贫困 + 极端的孤僻性格**

在巴黎，思特里克兰德穷困到差点饿死，但依旧顽固地为了画画活着。
 他行为粗鲁、自私、冷漠，却拥有惊人的艺术天赋和强烈的创造力。

### **3. 去塔希提岛寻找纯粹的艺术世界**

后来他前往南太平洋的塔希提岛，在那里生活淳朴自由，他娶了当地女子阿塔（Ata），并继续疯狂创作。

### **4. 完成伟大作品，死于麻风病**

在塔希提的草屋里，他创作了巨大的壁画，被认为是他一生的巅峰之作。
 但他感染麻风病，临死前命令妻子 **烧掉**壁画。他不在乎名声，只在乎创作时的那种“被艺术吞噬”的状态。

# **小说的核心思想**

- **月亮**象征梦想、艺术、超越世俗的理想。
- **六便士**象征现实生活、稳定、金钱、小利益。

大多数人都忙着寻找地上的六便士，而忽略了头顶的月亮。
 思特里克兰德则是一个**为了月亮，连六便士都不要的人**。

毛姆通过他探讨：

> 强大而纯粹的天才往往冷酷无情，而社会意义上的“好人”并不等于伟大的人。



毛姆深刻地探讨了“天才”与“人品”的关系。斯特里克兰德不是一个讨人喜欢的英雄，而是一个冷酷、自私、不负责任的混蛋。他抛家弃子，背叛恩人，对情人的死无动于衷。

毛姆借此提出一个尖锐的问题：**伟大的艺术是否源于高尚的灵魂？他的答案可能是否定的。**

他将斯特里克兰德的创作冲动描绘成一种原始的、魔鬼般的驱动力，一种“身不由己”的力量。就像被附体一样，他必须听从内心的召唤，为此他可以牺牲自己，也可以牺牲所有身边的人。这种力量是超越善恶的。毛姆不是在为这种行为辩护，而是在冷静地观察和呈现一个事实：人类历史上那些开辟新天地的天才，往往具有让普通人难以接受的破坏性力量。
