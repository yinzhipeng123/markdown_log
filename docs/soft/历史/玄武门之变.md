### 1️⃣ 背景

- **时间**：公元626年（唐朝贞观元年）。
- **地点**：长安的**玄武门**，唐朝皇宫北门。
- **主要人物**：
  - [ ] **李渊**：唐朝开国皇帝，唐高祖。
  - [ ] **李建成**：太子，李渊长子。
  - [ ] **李世民**：唐朝二子，后来的唐太宗。
  - [ ] **李元吉**：李建成弟弟，王世充支持者。
- **背景事件**：
  - [ ] 唐朝建立后，李渊的几个儿子在朝中权力斗争。
  - [ ] 太子李建成和弟弟李世民、李元吉之间矛盾激烈。
  - [ ] 李世民凭借在**平定隋末叛乱和统一战争中的战功**，威望极高，引发太子和弟弟的忌妒与防范。
  - [ ] 宫廷中传言太子有意加害李世民，李世民感到人身安全受到威胁。

------

### 2️⃣ 事件经过

**1.李世民预谋行动**：

- 626年7月2日，李世民决定先发制人。
- 他秘密召集自己的亲信和士兵，在**玄武门设伏**。

**2.玄武门之变**：

- 李建成与李元吉从宫中出来时，李世民发动攻击。
- 李建成和李元吉被当场杀死。
- 此事件一举消除了李世民的主要政治对手。

**3.事后**：

- 李世民控制朝政，并迫使李渊禅位。
- 李世民即位，称帝，是为**唐太宗**。

------

### 3️⃣ 事件意义

1. **政治斗争的高潮**：这是唐初王室内部斗争的关键事件，奠定了李世民的帝位。
2. **唐朝历史转折点**：李世民即位后，开启了“贞观之治”，政治清明、经济繁荣，成为中国历史上著名的治世之一。
3. **“先发制人”策略**：李世民凭借先见之明和果断行动，保全了自己的性命和权力，也展示了他卓越的政治手腕。
