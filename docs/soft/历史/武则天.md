

# 两人的身份背景

### **李治（唐高宗）**

- 唐太宗李世民的第九个儿子。
- 性格温和、身体较弱、政治手腕不如太宗强。
- 继位后年纪轻、体弱，朝政很快依赖武则天。

### **武则天**

- 原名武媚，14岁入宫做太宗的才人（妃子等级很低）。
- 李世民死后按惯例入感业寺为尼姑。
- 后来被太宗的儿子李治迎回宫，成为昭仪，后来当上皇后。

------

# 武则天与李治的感情与政治关系

### **1. 李治爱她是真的**

李治年轻时就对武则天有好感。太宗在世时，两人就互相倾心，但当时是不能发生关系的。
 李治即位后，把武则天从尼姑庵接回宫中，这在古代很少见，说明李治对她**非常喜欢**。

### **2. 武则天也确实深爱李治**

武则天后来虽然权力极大，但在李治在世期间，她从来没有越位称帝。
 她帮助李治处理政务，甚至在他病重时替他“临朝听政”，但始终保持“皇后”的身份。

这说明她真正尊重李治，很多决策都是帮李治巩固大唐。

------

# 两人如何共同掌权

### **1. 李治身体弱**

李治常年有头痛、眼疾、心脑血管问题，甚至开会都会晕过去。

因此他需要一个能帮他处理国家大事的人。

### **2. 武则天的政治能力极强**

她能：

- 迅速判断大局
- 抓官员腐败
- 整顿吏治
- 理解人心

于是形成了历史上著名的：
 👉 **“二圣临朝”**（皇帝+皇后一起上朝）。

武后做了很多大事，比如：

- 清理贪官（酷吏问题另说）
- 重用寒门官员
- 推动科举扩张
- 促进唐朝人口、经济持续增长

所以“李治+武则天”的时代其实是唐朝的**第二次盛世雏形**。

------

# 宫廷斗争是真实存在的

武则天上位不是平稳的，是宫廷斗争中赢出来的。

- 她将王皇后“逼退”（包括著名的“椒房酿祸”事件）。
- 她排挤长孙无忌、褚遂良等老臣。
- 她建立了非常强势的政治体系。

这些行为具有争议，但也确立了她和李治的稳定权力体系。

------

# 李治死后，武则天正式掌权

公元683年，高宗李治去世。
 按照正常流程应该是儿子李显（中宗）接班。

但李显太软，被武则天直接废掉，改立弟弟李旦（睿宗）。
 真正的实际权力完全在武则天手里。

最终在公元690年，她索性**自己称帝**，建立武周王朝，成为中国历史上唯一的女皇帝。







# 上官仪是谁？

上官仪（608—664）

- 唐朝著名文学家、政治家。
- 是唐高宗李治朝中“北门学士”的核心人物。
- 文采极高，负责起草诏书，是当时的“首席文案、首席秘书长”。

他的女儿就是后来的“才女宰相”——**上官婉儿**。

------

# 上官仪得罪了武则天

当时唐高宗李治对武则天的专权有点受不了，想废掉她。
 李治私下找上官仪，请他起草废后诏书。

上官仪文学水平极强，文笔又犀利，于是他写了一篇非常漂亮、非常正式的“废后诏书”草稿。

但这个秘密很快泄露了。

------

# 武则天雷霆反击

武则天听说上官仪“参与废后”，立即大怒。

她给上官仪扣了三条大罪：

1. **与皇帝密谋废后**
2. **制造政变**
3. **诏书“诽谤皇后”**

结果是：

📌 **公元664年，上官仪被武则天下令处死，全族连坐。**

上官婉儿当时刚出生几个月，就和母亲被打入掖庭（宫廷的“女奴官”）。
 她从婴儿时期就是宫廷罪奴。

------

# 最戏剧性的地方在这里：

上官仪被武则天杀了后，他的女儿上官婉儿却——

### ✨ 被武则天亲自提拔

✨ 成为武则天身边最重要的女官
 ✨ 后来还成为实际上的“女宰相”

可以说：

> **武则天杀了婉儿的父亲，却把婉儿培养成了朝廷最高级的女官之一。**

这是历史上最戏剧性的命运转折。

------

# 为什么武则天愿意重用上官婉儿？

因为上官婉儿从小天资聪慧，文笔极高，她继承了她父亲的文学才华。

武则天非常懂得用人才，尤其是文学高手（如上官仪、沈佺期、薛曜等）。

婉儿被赦免后，她用超强的文才把武则天写得“盛世明君”“圣母神皇”，武则天很喜欢。

从此婉儿一路高升。
