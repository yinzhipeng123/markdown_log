### 核心逻辑

不是说"你的理论是错的"，而是说："**按照你自己的标准，你自己的理论失败了。**"

这比外部批判有力得多，因为对方无法简单地说"我不接受你的前提"。

------

### 几种主要路径

**1. 自我反驳（Self-refutation）**

让对方的理论适用于自身，看它是否崩溃。

> 相对主义者说："所有真理都是相对的。" → 那么这句话本身是不是也是相对的？如果是，它对你不成立；如果不是，就有绝对真理了。

**2. 推论到荒谬（Reductio ad absurdum）**

接受对方的前提，严格推演，直到得出对方也必须拒绝的结论。

> 功利主义者说：道德就是最大化总体幸福。 → 严格推演：那么折磨一个无辜者来娱乐一亿人，在道德上是义务？ → 对方若拒绝，就必须修正理论。

**3. 挖掘内部矛盾**

找出理论内部两个承诺之间的张力，迫使对方取舍。

> 自由市场理论既主张"个人自由至上"，又要求"契约必须强制执行"。 → 但强制执行契约需要国家权力——这本身就是对自由的限制。 → 对方必须解释如何调和。

**4. 用对方的价值观倒打**

接受对方重视的价值，证明他们的政策实际上破坏了这个价值。

> 对方说：严刑峻法是为了保护公民安全。 → 用他们的数据和逻辑证明：严刑峻法实际上增加了社会暴力。 → 用他们的目标，否定他们的手段。

**5. 援引对方承认的例外，扩大化**

找到对方理论已经承认的例外情形，论证你的情况与之在结构上完全相同。

------

### 操作要点

- **先充分理解对方理论**，比他们自己更熟悉——否则你打的是稻草人
- **认真接受前提**，不要急于否定，要顺着走
- **找"承重墙"**：哪个概念是整个理论不可缺少的？从那里入手
- **保持礼貌**：内在批判越冷静，越有力，越显示你真的懂他们

------

### 哲学史上的经典范例

| 批判者                       | 被批判理论             | 方式                                           |
| ---------------------------- | ---------------------- | ---------------------------------------------- |
| 黑格尔批判康德               | 物自体不可知           | 既然完全不可知，你怎么知道它存在？             |
| 马克思批判黑格尔             | 精神推动历史           | 用黑格尔的辩证法，证明物质才是基础             |
| 维特根斯坦批判《逻辑哲学论》 | 他批判的是自己早期著作 | 该书最后说：懂我的人会明白，我的命题是无意义的 |
