



## 一个简单例子（80kg男）

举个例子：

### 高碳日

- 碳水 300g
- 蛋白 160g
- 脂肪 40g

### 中碳日

- 碳水 200g
- 蛋白 160g
- 脂肪 60g

### 低碳日

- 碳水 80g
- 蛋白 170g
- 脂肪 80g

------

## 碳循环的缺点

1️⃣ 比较复杂
 需要计算营养。

2️⃣ 情绪波动
 低碳日容易疲劳。

3️⃣ 不适合完全不运动的人。



## 最适合外卖人的一周碳循环

不用计算，只记主食。

| 日子 | 主食 |
| ---- | ---- |
| 周一 | 2碗  |
| 周二 | 半碗 |
| 周三 | 1碗  |
| 周四 | 半碗 |
| 周五 | 2碗  |
| 周六 | 1碗  |
| 周日 | 半碗 |







## 点单口诀

**肉 + 菜 + 控制主食**

例如：

高碳日

- 宫保鸡丁 + 米饭2碗
- 牛肉盖饭
- 鸡腿饭

中碳日

- 小碗米饭
- 加肉 + 菜

低碳日

- 水煮牛肉
- 干锅花菜
- 烤鱼
- 麻辣烫（不要粉）
