# 如何“重塑大脑”

------

## 1. 缺的不是智力，是“认知结构”

- 大多数人智力差距没你想的那么大
- 差的是：信息怎么组织、怎么调用
- 会思考的人，本质是“结构清晰”
- 不是更聪明，而是更有“模型”

------

## 2. 建立“模型思维”（最关键的一步）

- 遇到问题，不要只看表面
- 问：这是哪一类问题？有没有通用模型？
- 比如：激励问题、博弈问题、系统问题
- 一旦归类，你的判断会快很多

------

## 3. 强化“输入质量”，而不是输入数量

- 不是看得多就聪明
- 而是看“高密度、有结构”的内容
- 低质量信息（短视频碎片）会削弱思考能力
- 高质量信息会重塑你的判断框架

------

## 4. 训练“输出能力”

- 你能不能把一个问题讲清楚
- 决定了你是否真正理解
- 写下来
- 讲给别人听
- 自己复述一遍

------

## 5. 刻意训练“延迟反应”

- 人的默认模式是：快速情绪反应
- 聪明的人：先停一下，再判断
- 遇到问题先不下结论
- 多问一句“还有没有别的解释？”

------

## 6. 提升“抽象能力”

- 把具体问题，抽象成通用问题
- 例如：吵架 → 利益冲突 +沟通失效
- 例如：工作问题 → 激励机制问题
- 抽象之后，你能跨场景解决问题

------

## 7. 管理好身体（这个很多人忽视）

- 睡眠不足 → 直接降低认知能力
- 久坐、缺运动 → 大脑效率下降
- 血糖波动大 → 注意力不稳定
- 这些比“学习技巧”影响更大

------

## 8. 建立“正反馈系统”

- 大脑靠奖励学习
- 每次进步要有反馈（哪怕很小）
- 否则很难长期坚持
- 可以用：记录、打卡、阶段目标
