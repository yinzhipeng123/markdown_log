**Narcissistic Personality Disorder（自恋型人格障碍）\**是一种人格障碍，主要特征是\**过度自我中心、强烈需要他人赞美、缺乏共情能力**。它和普通的“自恋”“自信”不同，是一种持续、稳定的心理人格模式，会明显影响人际关系和生活。

------

## 一、NPD常见症状

根据 **American Psychiatric Association** 的诊断标准，常见表现包括：

### 1️⃣ 夸大的自我重要感

- 认为自己非常特殊、优秀
- 夸大成就或能力
- 期待别人把自己当成“很重要的人”

例如：
 觉得自己应该得到特殊待遇。

------

### 2️⃣ 强烈需要赞美

- 非常在意别人是否认可自己
- 如果没有得到赞美，会感到愤怒或沮丧
- 经常需要被关注

------

### 3️⃣ 缺乏共情能力

- 很难理解或在意别人的感受
- 对他人的痛苦或需求不敏感

------

### 4️⃣ 利用他人

- 把别人当作实现自己目标的工具
- 关系往往带有功利性

------

### 5️⃣ 嫉妒和被嫉妒感

- 容易嫉妒别人
- 也常认为别人嫉妒自己

------

### 6️⃣ 傲慢和优越感

- 表现出高高在上的态度
- 看不起别人

------

## 二、隐藏型 NPD（很多人不知道）

有一种叫 **脆弱型自恋**：

特点是：

- 外表看起来敏感、内向
- 内心仍然觉得自己“应该很特别”
- 对批评非常敏感
- 容易陷入羞耻和愤怒

------

## 三、NPD的形成原因

心理学认为通常与这些有关：

1. **童年过度溺爱**
   - 被不断夸奖“你是最特别的”
2. **童年忽视或羞辱**
   - 通过“自恋防御”保护自尊
3. **父母情感不稳定**
4. **遗传因素**

------

## 四、如何治疗

NPD治疗**比较困难，但不是无法改善**。

### 1️⃣ 心理治疗（最有效）

常见方法：

- **Psychodynamic Therapy（精神动力治疗）**
   探索童年与人格形成
- **Cognitive Behavioral Therapy（认知行为疗法）**
   改变扭曲认知
- **Schema Therapy（模式疗法）**
   专门用于人格障碍

治疗通常需要 **几年时间**。

------

### 2️⃣ 药物治疗

NPD本身没有特效药。

但如果同时出现：

- **Depression**
- **Anxiety Disorder**

医生可能会开药物治疗这些问题。

------

### 3️⃣ 自我改善（轻度情况）

如果只是有自恋倾向，可以通过：

- 练习 **共情能力**
- 接受批评
- 建立真实自我评价
- 减少对外界认可的依赖

------

## 五、重要区分

很多人会误解：

| 情况                 | 是否NPD |
| -------------------- | ------- |
| 自信                 | ❌       |
| 偶尔自我中心         | ❌       |
| 爱表现               | ❌       |
| 长期严重影响人际关系 | ✔       |

真正的NPD **在人群中约 1%–6%**。
