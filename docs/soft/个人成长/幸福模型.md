**PERMA** 是由积极心理学之父**马丁·塞利格曼**（Martin Seligman）提出的一个心理模型，旨在定义和测量“幸福”与“蓬勃人生”（Well-being）。

塞利格曼认为，幸福不仅仅是“快乐”这种单一的情绪，而是由五个核心元素构成的。只有当这五个方面都得到满足时，一个人才能真正达到身心健康、生活圆满的状态。

------

### PERMA 模型的五个核心元素

| **字母** | **代表元素**                    | **核心含义**                                                 |
| -------- | ------------------------------- | ------------------------------------------------------------ |
| **P**    | **Positive Emotion** (积极情绪) | 感受喜悦、感激、安宁、希望等情绪，这是幸福最直观的基础。     |
| **E**    | **Engagement** (投入/心流)      | 完全沉浸在某项活动中（如工作、运动、创作），体验到“心流”状态，此时时间感消失。 |
| **R**    | **Relationships** (人际关系)    | 与他人建立深刻、互助且有意义的连接。人类是社会动物，孤独是幸福的大敌。 |
| **M**    | **Meaning** (意义)              | 感觉自己属于并服务于比“自我”更大的事物。比如通过宗教、事业、家庭或理想找到存在的价值。 |
| **A**    | **Accomplishment** (成就)       | 为了获胜或达成目标而努力。这种“为了成就而成就”的追求，能带来自尊和效能感。 |

------

### 为什么这个模型很重要？

1. **从“修补”到“建设”：** 传统的心理学往往关注如何治疗抑郁和焦虑（从 -5 到 0），而 PERMA 关注如何让普通人生活得更精彩（从 0 到 +5）。
2. **可操作性：** 幸福不再是一个虚无缥缈的词。如果你感到最近状态不好，可以根据这五个维度进行自检：是最近太孤独了（R），还是工作找不到意义（M）？
3. **动态平衡：** 每个人对这五个元素的侧重不同。有人通过极度的成就感（A）获得幸福，有人则更看重人际关系的温暖（R）。
