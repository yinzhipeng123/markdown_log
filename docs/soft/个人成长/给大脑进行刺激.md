## 给大脑新鲜刺激



可以从简单开始：

- 每周去一个没去过的地方
- 做一件“没效率意义”的事（比如随便逛、尝试新餐厅）
- 运动选“刺激型”的（跑步冲刺、力量训练、对抗类）

👉 重点不是内容，而是：

> **让大脑感到“有变化、有未知”**



| 状态       | 内部情况               | 外部吸引力         |
| ---------- | ---------------------- | ------------------ |
| 满足型关系 | 有交流、有回应、有互动 | 外面的人普通       |
| 匮乏型关系 | 没交流、压抑、单一     | 外面的人很有吸引力 |
