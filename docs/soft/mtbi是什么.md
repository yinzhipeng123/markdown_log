

# 一、E vs I（能量来源）

这是 **你恢复精力的方式**。

| 类型       | 特点                   | 举例         |
| ---------- | ---------------------- | ------------ |
| **E 外向** | 从人与人互动中获得能量 | 聚会后更精神 |
| **I 内向** | 从独处中获得能量       | 独处后更精神 |

举个现实例子：

一天工作结束后：

- **E型人**：想找朋友吃饭聊天
- **I型人**：想自己回家安静待会

注意：
 **内向不等于社恐**
 很多演讲者其实是内向型。

# 二、S vs N（获取信息方式）

这是 **你看世界的方式**。

| 类型         | 特点                   | 举例         |
| ------------ | ---------------------- | ------------ |
| **S 感觉型** | 关注现实、细节、事实   | 记住具体数据 |
| **N 直觉型** | 关注趋势、意义、可能性 | 想象未来发展 |

举例：

看到一棵树：

- **S型**
   会注意：树多高、叶子形状
- **N型**
   会想到：这棵树象征生命、时间

再举个工作例子：

- **S型**：
   按流程做事
- **N型**：
   想新的方法

# 三、T vs F（做决策方式）

这是 **你如何判断事情**。

| 类型         | 特点               |
| ------------ | ------------------ |
| **T 思考型** | 用逻辑、规则判断   |
| **F 情感型** | 用价值观、人情判断 |

举例：

公司要裁员。

- **T型思维**

  > 谁绩效低就裁谁

- **F型思维**

  > 谁家庭困难要考虑

注意：

- **T不是冷血**
- **F不是不理性**

只是优先考虑不同。

# 四、J vs P（生活方式）

这是 **你处理事情的方式**。

| 类型         | 特点           |
| ------------ | -------------- |
| **J 判断型** | 喜欢计划和确定 |
| **P 知觉型** | 喜欢灵活和自由 |

举例：

出去旅游：

**J型人**

- 先做攻略
- 行程安排好

**P型人**

- 到了再看
- 随机探索

再比如：

工作任务：

- **J型**：提前完成
- **P型**：最后爆发完成

# 五、四个维度组合

每个人都会有一个组合，例如：

| 类型 | 含义                      |
| ---- | ------------------------- |
| INTJ | 内向 + 直觉 + 思考 + 计划 |
| ENFP | 外向 + 直觉 + 情感 + 随性 |
| ISTJ | 内向 + 现实 + 思考 + 计划 |

总共 **16种人格**。
