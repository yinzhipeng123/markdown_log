# 什么是卷积



### **假设：一张 5×5 的灰度图像**

每个数字代表像素亮度（0-9）：

```
图像：
1 2 3 0 1
4 5 6 1 2
7 8 9 0 1
1 2 3 4 5
0 1 0 2 3
```

------

### **卷积核（3×3）**

这是一个简单的卷积核，用来检测“边缘”：

```
卷积核：
0 1 0
1 -4 1
0 1 0
```

------

### **卷积操作**

1. 卷积核在图像上从左上角开始滑动。
2. 每次覆盖 3×3 区域，把对应的元素相乘，然后加起来，得到一个数字（特征图的一个元素）。
3. 举个例子：卷积核覆盖图像左上角的 3×3 区域：

```
图像覆盖区域：
1 2 3
4 5 6
7 8 9
```

计算卷积结果：

```
(1*0) + (2*1) + (3*0) +
(4*1) + (5*-4) + (6*1) +
(7*0) + (8*1) + (9*0)
= 0 + 2 + 0 + 4 - 20 + 6 + 0 + 8 + 0
= 0
```

所以特征图的左上角对应位置就是 **0**。

------

### **继续滑动**

卷积核向右移动一格，继续计算，直到整张图都被扫描，得到 **整个特征图**。

------

### **直观理解**

- 图像 → 卷积层 → 特征图
- 卷积核像“扫描眼睛”，每次只看图像一小块，提取某种特征（边缘、纹理）。
- 卷积层提取的是**局部信息**，而且通过不同卷积核可以得到不同特征。
