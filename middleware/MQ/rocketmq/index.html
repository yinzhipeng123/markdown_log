    дёҖгҖҒзҺҜеўғеҮҶеӨҮ

    1. ж“ҚдҪңзі»з»ҹпјҡ

    2. Linux жҳҜжҺЁиҚҗзҡ„зҺҜеўғпјҢдҪҶд№ҹж”ҜжҢҒ WindowsгҖӮ

    3. дҫқиө–зҺҜеўғпјҡ

    4. JDK 1.8 жҲ–жӣҙй«ҳзүҲжң¬пјҲJava зҺҜеўғпјүгҖӮ
    5. й…ҚзҪ® JAVA_HOME зҺҜеўғеҸҳйҮҸгҖӮ

    6. зЎ®дҝқжңҚеҠЎеҷЁзҪ‘з»ңйҖҡз•…пјҢз«ҜеҸЈжңӘиў«еҚ з”ЁпјҲй»ҳи®Ө 9876 дёә NameServer з«ҜеҸЈпјҢ10911 е’Ң 10909 дёә Broker дҪҝз”Ёзҡ„з«ҜеҸЈпјүгҖӮ

    7. дёӢиҪҪ RocketMQпјҡ

    8. д»Һ RocketMQ е®ҳж–№ GitHub д»“еә“ дёӢиҪҪзЁіе®ҡзүҲжң¬зҡ„дәҢиҝӣеҲ¶еҢ…жҲ–жәҗз ҒгҖӮ

    9. и§ЈеҺӢе№¶й…ҚзҪ®пјҡ

    10. е°ҶдёӢиҪҪзҡ„ RocketMQ и§ЈеҺӢеҲ°жҢҮе®ҡзӣ®еҪ•гҖӮ

    дәҢгҖҒеҚ•жңәйғЁзҪІ

    1. еҗҜеҠЁ NameServer

    NameServer жҳҜ RocketMQ зҡ„жңҚеҠЎеҸ‘зҺ°з»„д»¶пјҢиҙҹиҙЈз®ЎзҗҶ Broker дҝЎжҒҜгҖӮ

    # иҝӣе…Ҙ RocketMQ зӣ®еҪ•
cd rocketmq-all-<version>/bin

# еҗҜеҠЁ NameServer
nohup sh mqnamesrv > logs/nameserver.log 2>&1 &

    2. еҗҜеҠЁ Broker

    Broker жҳҜж¶ҲжҒҜзҡ„еӯҳеӮЁе’ҢиҪ¬еҸ‘з»„д»¶гҖӮ

    # еҗҜеҠЁ Broker
nohup sh mqbroker -n localhost:9876 > logs/broker.log 2>&1 &

    еҸӮж•°иҜҙжҳҺпјҡ - -n localhost:9876пјҡжҢҮе®ҡ NameServer ең°еқҖгҖӮ - logs/broker.logпјҡж—Ҙеҝ—ж–Үд»¶и·Ҝеҫ„гҖӮ

    3. йӘҢиҜҒжңҚеҠЎ

    дҪҝз”Ё jps жҹҘзңӢжҳҜеҗҰжңү NamesrvStartup е’Ң BrokerStartup дёӨдёӘ Java иҝӣзЁӢиҝҗиЎҢгҖӮ

    jps

    дёүгҖҒйӣҶзҫӨйғЁзҪІпјҲжҺЁиҚҗз”ҹдә§зҺҜеўғдҪҝз”Ёпјү

    1. йғЁзҪІжһ¶жһ„

    йӣҶзҫӨйғЁзҪІдёҖиҲ¬еҢ…жӢ¬пјҡ - еӨҡдёӘ NameServerпјҡжҸҗй«ҳ NameServer зҡ„еҸҜз”ЁжҖ§гҖӮ - еӨҡдёӘ Brokerпјҡ - Master-Slave жЁЎејҸпјҡе®һзҺ°й«ҳеҸҜз”ЁпјҲжҺЁиҚҗпјүгҖӮ - еӨҡ Master жЁЎејҸпјҡе®һзҺ°жӣҙй«ҳзҡ„еҗһеҗҗйҮҸгҖӮ

    2. й…ҚзҪ® NameServer

    еңЁжҜҸеҸ° NameServer иҠӮзӮ№дёҠжү§иЎҢд»ҘдёӢжӯҘйӘӨпјҡ 

    nohup sh mqnamesrv > logs/nameserver.log 2>&1 &

    3. й…ҚзҪ® Broker

    дёәжҜҸдёӘ Broker иҠӮзӮ№еҲӣе»әзӢ¬з«Ӣзҡ„й…ҚзҪ®ж–Үд»¶пјҢдҫӢеҰӮ broker-a.properties е’Ң broker-b.propertiesпјҢзӨәдҫӢеҰӮдёӢпјҡ

    Master й…ҚзҪ®ж–Үд»¶пјҲbroker-a.propertiesпјүпјҡ 

    brokerClusterName=DefaultCluster
brokerName=broker-a
brokerId=0 # Master
namesrvAddr=<nameserver1_ip>:9876;<nameserver2_ip>:9876
storePathRootDir=/path/to/store
storePathCommitLog=/path/to/commitlog

    Slave й…ҚзҪ®ж–Үд»¶пјҲbroker-a-s.propertiesпјүпјҡ 

    brokerClusterName=DefaultCluster
brokerName=broker-a
brokerId=1 # Slave
namesrvAddr=<nameserver1_ip>:9876;<nameserver2_ip>:9876
storePathRootDir=/path/to/store-slave
storePathCommitLog=/path/to/commitlog-slave

    еҗҜеҠЁ Brokerпјҡ 

    nohup sh mqbroker -c conf/broker-a.properties > logs/broker-a.log 2>&1 &
nohup sh mqbroker -c conf/broker-a-s.properties > logs/broker-a-s.log 2>&1 &

    еӣӣгҖҒжөӢиҜ•йғЁзҪІ

    1. еҸ‘йҖҒж¶ҲжҒҜпјҡ дҪҝз”Ё RocketMQ жҸҗдҫӣзҡ„е·Ҙе…·еҸ‘йҖҒжөӢиҜ•ж¶ҲжҒҜгҖӮ 

      sh tools.sh org.apache.rocketmq.example.quickstart.Producer

    2. жҺҘж”¶ж¶ҲжҒҜпјҡ жҺҘж”¶жөӢиҜ•ж¶ҲжҒҜгҖӮ 

      sh tools.sh org.apache.rocketmq.example.quickstart.Consumer

    дә”гҖҒеҸҜйҖүй…ҚзҪ®пјҲдјҳеҢ–е’Ңй«ҳеҸҜз”Ёпјү

    1. ж—Ҙеҝ—й…ҚзҪ®

    дҝ®ж”№ logback_broker.xml е’Ң logback_namesrv.xml дёӯзҡ„ж—Ҙеҝ—зә§еҲ«е’Ңи·Ҝеҫ„гҖӮ

    2. JVM еҸӮж•°и°ғж•ҙ

    ж №жҚ®жңҚеҠЎеҷЁй…ҚзҪ®и°ғж•ҙ runserver.sh е’Ң runbroker.sh дёӯзҡ„ JVM еҸӮж•°пјҡ 

    JAVA_OPT="${JAVA_OPT} -server -Xms8g -Xmx8g -Xmn4g"

    3. иҝҗз»ҙе·Ҙе…·

    RocketMQ жҸҗдҫӣ Web жҺ§еҲ¶еҸ°пјҢеҸҜд»Ҙж–№дҫҝең°з®ЎзҗҶе’Ңзӣ‘жҺ§йӣҶзҫӨпјҡ - дёӢиҪҪпјҡRocketMQ Dashboard - йғЁзҪІе№¶и®ҝй—®гҖӮ

    е…ӯгҖҒеёёи§Ғй—®йўҳ

    1. NameServer ж— жі•иҝһжҺҘпјҡ
    2. жЈҖжҹҘ 9876 з«ҜеҸЈжҳҜеҗҰејҖж”ҫгҖӮ

    3. зЎ®дҝқ namesrvAddr й…ҚзҪ®жӯЈзЎ®гҖӮ

    5. зЎ®дҝқ storePathCommitLog е’Ң storePathRootDir жңүи¶іеӨҹзҡ„жқғйҷҗгҖӮ

    6. жЈҖжҹҘ JVM еҸӮж•°жҳҜеҗҰйҖӮй…ҚеҪ“еүҚзЎ¬д»¶гҖӮ
